## WP-Admin
Installation:
```
composer install
```

### Configuration
Update paths in 
```
./config/app.ini
```


### Single Backup and Update for Wordpress
```
./wp-admin backup [path_to_wordpress]
```
```
./wp-admin update [path_to_wordpress]
```

### Batch Backup with CSV file
```
./wp-admin backupsource
```
